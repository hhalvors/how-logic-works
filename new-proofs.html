How Logic Works: Exercises

Hans Halvorson

New Proofs from Old

Exercise 4.1 Which of the following formulas is a substitution instance of the formula P → ¬Q? In cases where you answer affirmatively, show how to reconstrue P and Q to get the resulting formula.

  1. ¬Q → ¬P

  2. (P→¬Q) → R

  3. (P→¬Q) → ¬(P→¬Q)

Aside. A formula ψ is a substitution instance of another formula ϕ if and only if ϕ provides a set of instructions for disassembling ψ into subformulas. For example, if ϕ is of the form

And(IfThen(P,Q),Not(R))

for atomic sentences P, Q, R, then any substitution instance of ϕ is of the form

And(IfThen(X,Y),Not(Z))

where X, Y and Z are well-formed formulas.

Exercise 4.2 Do you think the following claims are true or false? Explain your answers.

  1. If ϕ is not provable, then no substitution instance of ϕ is provable.

  2. The sentence (P∧Q) → R has a substitution instance that can be proven.

Exercise 4.3 Prove the following sequents. You may invoke cut with any named sequent that you have already proven (e.g. excluded middle, positive paradox, negative paradox).

  1. P ↔︎ Q ⊢ Q ↔︎ P

  2. ⊢ (Q→P) ∨ (P→R)

  3. P ↔︎ Q  ⊣ ⊢ (P∧Q) ∨ (¬P∧¬Q)

  4. ¬(P↔︎P) ⊢ Q ∧ ¬Q

  5. P ↔︎ Q, Q ↔︎ R ⊢ P ↔︎ R

  6. ¬(P↔︎Q)  ⊣ ⊢ (P↔︎¬Q)

  7. ⊢ (P↔︎Q) ∨ (P↔︎R) ∨ (Q↔︎R)
